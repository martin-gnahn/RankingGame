# Ticket Workflow

RankingGame ticket and planning markdown files are stored outside this repository, in the sibling planning workspace:

```text
../RankingGamePlanning/Plans
```

This repository should only contain durable project documentation and implementation-relevant knowledge. Raw ticket
descriptions, daily plans, sync reports, and temporary planning notes belong in the external planning workspace.

The external planning workspace is a local working context, not the product source of truth. GitHub Issues/Projects or
the configured ticket system remain authoritative for ticket state.
